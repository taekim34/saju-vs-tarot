# Plan: engine-upgrade (사주/타로 엔진 전문가 수준 업그레이드)

> Created: 2026-03-07
> Feature: engine-upgrade
> Status: Plan

## 1. Overview

사주 명인(50년)과 집시 할머니(30년)가 평가해도 자신 있을 정도의 논리 구조로 양 엔진을 업그레이드한다.
**양쪽 모두 90점 수준으로 올려야 대등한 배틀이 가능하다.**

### 현재 전문가 평가

| 엔진 | 현재 점수 | 목표 점수 | 핵심 문제 |
|------|-----------|-----------|-----------|
| 사주 (saju.js) | ~35/100 | **~90/100** | 지지 십성 버그, 지장간 미활용, 충합 미구현, 세운/대운 없음, 절기 부정확 |
| 타로 (tarot.js) | ~75/100 | **~90/100** | 역방향 50% 비현실적, 카드 관계 분석 전무, 패턴 미감지 |
| AI 프롬프트 | ~70/100 | **~90/100** | 엔진 분석 데이터를 프롬프트에 충분히 전달하지 못함 |

## 2. Scope

### In Scope
- saju.js: 버그 수정 + 지장간 + 세운 + 대운 + 충/합 + 12운성 + 절기 정밀화 (8개 전부)
- tarot.js: 역방향 확률 + 패턴 분석 + 원소 상성 + buildSummary 강화 (6개 전부)
- ai-interpreter.js: 강화된 엔진 데이터를 프롬프트에 반영

### Out of Scope
- UI 변경 (명식표 렌더링은 이미 완료)
- 타로 Celtic Cross 등 새 스프레드 추가

#### S1. 지지 십성 음양 버그 (Critical)
```
현재 (버그): const sameYinYang = targetYinYang ? ... : true;
→ 지지는 항상 true가 되어 편재/편관/편인만 나옴 (정재/정관/정인이 절대 안 나옴)

수정: 지지의 음양을 jiji.json의 yin_yang 데이터에서 가져와 비교
→ JIJI_YINYANG 매핑 추가 또는 jijiData에서 조회
```

#### S2. 세운(歲運) 추가
```
현재년도의 천간/지지를 계산하여:
- 세운 천간 vs 일간 → 십성 관계 산출 (올해의 에너지가 나에게 미치는 영향)
- 세운 지지 vs 사주 지지 → 충/합 판단
- 결과를 summary에 추가하여 AI에 전달

예) 2026년 = 丙午년
→ 일간이 甲이면: 丙은 식신(내 재능이 빛나는 해), 午는 화기운으로 지장간 丁己가 상관+정재
```

#### S3. 지장간 활용
```
jiji.json에 이미 jijanggan 데이터가 있음 (예: 寅 = ["甲","丙","戊"])
→ 지장간의 천간들을 오행으로 변환하여 "숨은 오행 분포" 계산
→ 기존 오행 분포(표면: 천간+지지 8자) + 지장간 오행(심층) 두 레이어 분석
→ 지장간 십성도 산출하여 "숨은 십성" 정보 제공

예) 寅의 지장간 [甲,丙,戊] → 목+화+토가 숨어있음
→ 표면에 화가 없어도 지장간에 丙이 있으면 "잠재된 화기운"
```

#### S4. 충(冲) 판단
```
6충 테이블:
子-午, 丑-未, 寅-申, 卯-酉, 辰-戌, 巳-亥
→ 사주 내 4개 지지끼리 충이 있는지 검사 (내부 충돌)
→ 세운 지지와 사주 지지 간 충도 검사 (올해의 충격)

의미:
- 子午충: 수화 충돌 → 감정과 이성의 갈등
- 寅申충: 목금 충돌 → 직업/방향의 변화
- 卯酉충: 목금 충돌 → 대인관계 변동
- 辰戌충: 토토 충돌 → 재산/부동산 변동
- 丑未충: 토토 충돌 → 가정 내 갈등
- 巳亥충: 화수 충돌 → 건강/학업 주의
```

#### S5. 삼합/육합
```
삼합 (3개 지지가 모이면 강력한 오행 국 형성):
  申子辰 → 수국(水局): 지혜와 유연함의 에너지
  寅午戌 → 화국(火局): 열정과 행동의 에너지
  巳酉丑 → 금국(金局): 결단력과 실행의 에너지
  亥卯未 → 목국(木局): 성장과 인자함의 에너지

육합 (2개 지지가 만나면 새로운 오행 생성):
  子丑 → 토, 寅亥 → 목, 卯戌 → 화, 辰酉 → 금, 巳申 → 수, 午未 → 토/화

→ 사주 내 3~4개 지지가 삼합을 이루는지 검사 (완전삼합 or 반합)
→ 인접 지지끼리 육합이 되는지 검사
→ 세운 지지가 사주 지지와 합을 이루는지도 검사
```

#### S6. 대운(大運) 계산
```
10년 단위의 큰 운의 흐름:
- 남자 양년생 / 여자 음년생 → 순행 (월주에서 앞으로 진행)
- 남자 음년생 / 여자 양년생 → 역행 (월주에서 뒤로 진행)

대운 시작 나이 계산:
- 생일에서 가장 가까운 절기까지의 일수 ÷ 3 = 대운 시작 나이

현재 나이로 현재 대운 간지 산출:
- 현재 대운의 천간/지지 → 십성 관계 분석
- 대운이 바뀌는 시점 정보 제공

예) 월주가 丙寅이고 순행이면:
  1대운: 丁卯, 2대운: 戊辰, 3대운: 己巳, ...
→ 현재 나이에 해당하는 대운의 에너지를 AI에 전달
```

#### S7. 12운성
```
일간의 12단계 에너지 상태 (생로병사의 순환):
장생→목욕→관대→건록→제왕→쇠→병→사→묘→절→태→양

10x12 매트릭스 (각 천간이 각 지지에서 어떤 단계인지):
甲: 亥(장생) 子(목욕) 丑(관대) 寅(건록) 卯(제왕) 辰(쇠) 巳(병) 午(사) 未(묘) 申(절) 酉(태) 戌(양)
乙: 午(장생) 巳(목욕) 辰(관대) 卯(건록) 寅(제왕) 丑(쇠) 子(병) 亥(사) 戌(묘) 酉(절) 申(태) 未(양)
... (나머지 8개 천간도 동일 구조)

→ 각 지지에서 일간이 어떤 단계인지 매핑
→ 제왕/건록이면 매우 강, 사/묘/절이면 매우 약
→ 신강/신약 판정의 정밀도가 크게 향상됨
```

#### S8. 정밀 절기 테이블
```
현재: 고정값 근사 (예: 입춘 = 2/4)
→ 매년 절기 시각이 다름 (최대 ±1~2일 오차)

개선 방안 (현실적 접근):
- 2020~2030년 범위의 주요 절기(12개) 시작 날짜를 테이블로 보유
- 연도별 절기 테이블 참조 방식

12절기 (월 시작 기준):
입춘(1월=寅), 경칩(2월=卯), 청명(3월=辰), 입하(4월=巳),
망종(5월=午), 소서(6월=未), 입추(7월=申), 백로(8월=酉),
한로(9월=戌), 입동(10월=亥), 대설(11월=子), 소한(12월=丑)

→ 연도별 테이블이 없으면 기존 고정값 fallback
→ 최소한 2024~2028년 정도는 정밀 테이블 보유
```

### 3.2 타로 엔진 개선 — 6개 항목 전체

| # | 개선항목 | 효과 | 난이도 | 수정 파일 |
|---|---------|------|--------|-----------|
| T1 | **역방향 확률 35%** | 현실적 리딩 | ⭐ | tarot.js |
| T2 | **수트 패턴 분석** | 원소 에너지 감지 | ⭐⭐ | tarot.js |
| T3 | **숫자 패턴 분석** | 수비학적 메시지 | ⭐⭐ | tarot.js |
| T4 | **메이저/마이너 비율** | 운명 vs 일상 판단 | ⭐ | tarot.js |
| T5 | **원소 상성(Elemental Dignity)** | 카드 간 "대화" | ⭐⭐ | tarot.js |
| T6 | **코트 카드 감지** | 인물 에너지 분석 | ⭐ | tarot.js |

#### T1. 역방향 확률 조정
```
현재: Math.random() < 0.5  (50%)
수정: Math.random() < 0.35  (35%)
→ 실제 타로 리더의 역방향 비율에 가깝게
```

#### T2. 수트 패턴 분석
```
같은 수트(완드/컵/소드/펜타클)가 2장 이상이면 감지:
- 수트별 원소 에너지: 완드=불(열정/행동), 컵=물(감정/직관), 소드=바람(사고/갈등), 펜타클=흙(물질/안정)
- "컵 3장 → 감정과 직관의 에너지가 압도적"
- 수트 0인 원소도 "부재 에너지"로 의미 부여 ("흙 원소 부재 → 현실적 기반이 약함")
```

#### T3. 숫자 패턴 분석
```
같은 숫자가 2장 이상이면 수비학적 의미 부여:
- 1(에이스): 새로운 시작의 에너지 증폭
- 2: 선택과 균형의 테마
- 3: 창조와 확장의 에너지
- 4: 안정과 구조화
- 5: 변화와 도전
- 6: 조화와 책임
- 7: 내면 탐구와 영적 성장
- 8: 힘과 성취
- 9: 완성 직전, 지혜
- 10: 순환의 완성, 새로운 시작의 문턱
→ "7이 2장 = 내면 탐구의 시간이 겹쳐 강조됨"
```

#### T4. 메이저/마이너 비율
```
메이저 아르카나 비율 분석:
- 3장 중 2장 이상 메이저 → "운명의 큰 전환기, 우주가 강력한 메시지를 보내고 있음"
- 5장 중 3장 이상 메이저 → "영적 메시지가 매우 강력, 인생의 중대한 갈림길"
- 전부 마이너 → "일상적 차원의 변화, 스스로의 선택이 더 중요"
- 전부 메이저 → "극히 드문 조합, 운명이 강력하게 개입하는 시기"
```

#### T5. 원소 상성 (Elemental Dignity)
```
인접 카드의 원소 관계 (Golden Dawn 전통):
- 우호(Friendly): 불+바람, 물+흙 → 에너지가 서로 강화됨
- 적대(Hostile): 불+물, 바람+흙 → 긴장과 갈등, 에너지 약화
- 중립(Neutral): 같은 원소 → 에너지 집중/과잉
- 메이저 아르카나는 element 필드로 원소 판별

→ 각 인접 카드 쌍의 관계를 분석하여 "카드 간 대화" 생성
→ "과거 위치의 컵(물)과 현재 위치의 완드(불)가 적대 → 감정과 행동 사이의 갈등"
```

#### T6. 코트 카드 감지
```
Page/Knight/Queen/King 카드 존재 시:
- Page: 메시지, 소식, 학습 에너지 (젊은이의 영향)
- Knight: 빠른 행동, 변화의 바람 (활동적 인물의 등장)
- Queen: 내면의 성숙, 수용적 지혜 (조언자의 역할)
- King: 외적 권위, 리더십 (결정권자의 영향)
→ 코트 카드가 있으면 "주변 인물의 영향" 또는 "내면의 특정 에너지" 해석 추가
→ 같은 수트의 코트 카드 + 숫자 카드 → 해당 원소의 인격화
```

## 4. Implementation Order

```
Phase 1 — 버그 수정 + 기초 (사주 35→50, 타로 75→80)
  S1(지지음양버그) → T1(역방향35%) → T4(메이저비율) → T6(코트카드)

Phase 2 — 핵심 분석 (사주 50→70, 타로 80→85)
  S2(세운) → S3(지장간) → T2(수트패턴) → T3(숫자패턴) → T5(원소상성)

Phase 3 — 고급 분석 (사주 70→85)
  S4(충) → S5(삼합/육합) → S7(12운성) → S8(절기정밀화)

Phase 4 — 대운 + 통합 (사주 85→90, 타로 85→90)
  S6(대운계산) → S-buildSummary 전면강화 → T-buildSummary 전면강화 → P1~P4(프롬프트)
```

## 5. Success Criteria

| 항목 | 기준 |
|------|------|
| 사주 엔진 | 8개 개선 전부 구현, 지지 십성 정확도 확인, 세운/대운/지장간/충합/12운성 데이터가 summary에 포함 |

| AI 해석 | 강화된 데이터 기반으로 사주/타로 양쪽 모두 더 깊이 있는 해석 생성 확인 |

| 배틀 밸런스 | 3라운드 테스트 시 사주/타로 해석 품질이 비슷한 수준 |

## 6. Risk

| 리스크 | 대응 |
|--------|------|
